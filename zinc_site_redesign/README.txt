These are the scripts for the protocol capture of Khare, Kipnis, Greisen et al. Nature Chemical Biology (2012).  See the zinc_site_redesign demo in the demos repository for detailed usage instructions.

Note that some scripts require the SciPy package (https://www.scipy.org/).

Moved to tools by Vikram K. Mulligan (vmullig@uw.edu) on 21 June 2016, as part of the 2016 Documentation XRW.
